How to configure a website

Step1:Get the equipment

Installing all the tools needed to develop the software
1.Git:It serves as a control system for tracking changes to the code base, collaborating, managing project versions, which allows us to clone, commit, and push changes to GitHub.
2.Github:For our website,we utilize the github service.
3.Github Desktop: To move or push our coding from local to GitHub, we used github desktop.
4.VScode:We used the Visual Studio code to write down our documents.
5.Markdown and Html language:to write our document.
6.CSS: Corresponds with html. Used to style the website.

Step2:Github creates a repository

1. To create a warehouse, click "new".
2.Name it in step 1, set "public" in step 2, and click "README" in step 3. Step 4 Click "create respository" to create the respository.
3. Click on "setting","collaborators"
4. Click "add people" to add members.

Step3:Clone and upload

On your GitHub homepage, click on your profile picture and select "Your repositories" to view all of your repositories. -Click on the warehouse you want to clone.
Click new to create a warehouse
Set the warehouse name and make it public, then click on create
Generate web page addresses on the settings page
Select the repository to be cloned on Githup software
Select the location of the clone warehouse folder for cloning
Find the folder where the library is located and upload the web code file
Build web code
